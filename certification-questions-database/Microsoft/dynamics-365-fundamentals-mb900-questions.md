## Question 4 - Topic 1
**Question Type:** Multiple Choice

**Question Text:**  
You need to select a Dynamics 365 for Customer Engagement application to help your organization effectively track, manage, and deliver project-based services.  
Which application should you select?

**Answer Choices:**  
- A. Customer Service
- B. Field Service
- C. Project Service automation
- D. Sales

**Correct Answer:** C - Project Service automation

**Explanation:**  
Project Service automation (PSA) is specifically designed for organizations that deliver project-based services, providing tools for project planning, resource management, time and expense tracking, and project delivery.

- https://docs.microsoft.com/en-us/dynamics365/customer-engagement/project-service/overview
