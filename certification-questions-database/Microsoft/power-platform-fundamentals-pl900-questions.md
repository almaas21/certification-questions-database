## Question 1 - Topic 1
**Question Type:** Multiple Choice  

**Question Text:**  
This question requires that you evaluate the underlined text to determine if it is correct. You have a Power Apps app. You create a new version of the app and then publish the new version. A customer goes through the process of restoring the previous version of the app. In the Version tab for the app, you will see two versions of the app. Review the underlined text. If it makes the statement correct, select `No change is needed.` If the statement is incorrect, select the answer choice that makes the statement correct.

**Underlined Text:** two

**Answer Choices:**  
- A. No change is needed.
- B. one
- C. three
- D. four

**Correct Answer:** C - three

**Explanation:**  
When you publish a new version and then restore a previous version, the Version tab will show three versions: the current version, the restored version, and any other versions that existed.
