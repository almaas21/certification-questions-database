## Question 20 - Topic 1
**Question Type:** Multiple Choice

**Question Text:**  
Your company has a Microsoft Office 365 subscription. As an administrator for this subscription, you have been tasked with recommending a solution that prohibits users from copying corporate information from managed applications installed on unmanaged devices. Which of the following should you recommend?

**Answer Choices:**  
- A. Windows Virtual Desktop
- B. Microsoft Intune
- C. Windows AutoPilot
- D. Azure AD Application Proxy

**Correct Answer:** B - Microsoft Intune

**Explanation:**  
Microsoft Intune is the correct solution for this scenario. It provides Mobile Device Management (MDM) and Mobile Application Management (MAM) capabilities that can control how corporate data is accessed and shared, including preventing data copying on unmanaged devices.

**Community Voting:**  
- B: 75%
- A: 25%

**Community Discussion:**  
Intune allows organizations to protect data on both managed and unmanaged devices by isolating corporate data from personal data, with policies that prevent unauthorized data copying and sharing.
